# Markdown to HTML Converter

A freeCodeCamp certification project that converts Markdown text into HTML in real time.

## What it does

Type Markdown into the input area and the app instantly shows you the raw HTML output alongside a live rendered preview — all side by side.

## Files

- `index.html` — page structure with three panels: input, raw HTML output, and preview
- `styles.css` — layout and styling
- `script.js` — conversion logic

## Technologies

HTML, CSS, JavaScript
